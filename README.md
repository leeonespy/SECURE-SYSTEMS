# SECURE-SYSTEMS
Sistema de Autenticação e Monitoramento de Usuários desenvolvido em Python. O sistema valida login com base em dados armazenados em JSON, registra tentativas de acesso em arquivos TXT e JSON, gera relatórios de sucesso e falha por usuário e emite alertas quando há múltiplas tentativas inválidas, simulando um controle básico de segurança.
